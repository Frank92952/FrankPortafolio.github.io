Sobre mí

A lo largo de mi carrera Universitaria he adquirido una amplia experiencia en la creación de sitios web y aplicaciones web que no solo tienen un diseño atractivo, sino que también están optimizados para la experiencia del usuario y la conversión. Habilidades

+

CSS:3

CSS (Cascading Style Sheets) es un lenguaje utilizado para definir la presentación y el aspecto visual de un documento HTML. Es una tecnología fundamental en el desarrollo web que permite controlar la apariencia de los elementos en una página, como el diseño, los colores, las fuentes, los espacios, los tamaños y otros estilos.

+

HTML:5

HTML (HyperText Markup Language) es un lenguaje de marcado utilizado para crear la estructura y el contenido de páginas web. Es el lenguaje fundamental en el desarrollo web y proporciona la base sobre la cual se construyen los sitios y aplicaciones web HTML utiliza etiquetas o elementos para marcar diferentes partes de un documento.

+

JavaScript

JS se centran en la estructura y presentación de una página web, respectivamente, JavaScript se utiliza para crear comportamientos dinámicos. Esto significa que se ejecuta en el navegador web del usuario y permite interactuar con elementos de la página, responder a eventos, realizar cálculos, manipular datos y comunicarse con servidores web.

+

BOOTSTRAP

Bootstrap es un framework front-end de código abierto ampliamente utilizado en el desarrollo web. Proporciona un conjunto de herramientas, componentes y estilos predefinidos que facilitan la creación de interfaces web modernas

+

TAILWIND

Tailwind CSS es un framework de diseño de código abierto utilizado para construir interfaces web personalizadas y receptivas. Se basan en componentes predefinidos, Tailwind CSS se centra en proporcionar una amplia variedad de utilidades CSS de bajo nivel que se pueden combinar para crear estilos y diseños personalizados.

+

JAVA

Java es un lenguaje de programación de propósito general, orientado a objetos y de alto nivel.Java se ha vuelto extremadamente popular y es ampliamente utilizado en una amplia variedad de aplicaciones, desde desarrollo de software empresarial hasta desarrollo de aplicaciones móviles y embebidas.
